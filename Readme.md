
### Тестовое задание: Проект№6. Козловская Дарья Константинова, 4 курс, ТУСУР, Информатика и вычислительная техника.

# C++
Создаёт Unix-сокет по пути `/tmp/unix_sock`, ожидает подключения, принимает идентификатор от Rust-программы, записывает его в лог с датой и временем, после чего отправляет свой идентификатор `SOCKET_SERVER_ON_CPP_HI`.

# Rust
Подключается к Unix-сокету `/tmp/unix_sock`, отправляет свой идентификатор `SOCKET_CLIENT_ON_RUST_HI`, принимает ответ от C++ программы, записывает его в лог с датой и временем.

- Установлен `g++` для компиляции C++
- Установлен `cargo` и `rustc` для компиляции Rust
- Соответсвенные лог-файлы находятся внутри архивов директорий программ.
